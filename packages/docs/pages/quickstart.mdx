# Quick Start

## CDN Installation

You can include the script using any of these CDN links:

[View on jsDelivr](https://www.jsdelivr.com/package/npm/@magape-track/core)

```html
<!-- jsDelivr CDN -->
<script src="
https://cdn.jsdelivr.net/npm/@magape-track/core@0.1.0/dist/index.min.js
"></script>

```

## NPM Installation

```bash
npm install @magape-track/core
```

## Basic Usage

Add the following code to your HTML file:

```html
<!-- Include user activity tracking script -->
<script src="
https://cdn.jsdelivr.net/npm/@magape-track/core@0.1.0/dist/index.min.js
"></script>


<!-- Initialize configuration -->
<script>
    const activityMonitor = new UserActivityMonitor({
        // if 10 seconds no activity, consider inactive
        noActivityThreshold: 10000,
        
        // sync data every minute
        sendInterval: 60000,

        // data reporting endpoint (if not provided, it will use the default endpoint to magape)
        endpoint: '/api/activity-log',
    });
</script>
```

## Configuration Options

| Parameter | Required | Description | Default Value |
|-----------|----------|-------------|---------------|
| gameName | Yes | Game name | null |
| noActivityThreshold | No | Inactivity detection threshold | 10000ms |
| sendInterval | No | Data reporting interval | 30000ms |
| endpoint | No | Data reporting API endpoint | /api/activity-log |
| storageKey | No | Local storage key name | user_activity_data |
| gameId | No | Game ID | null |
| ipEndpoint | No | IP lookup endpoint | https://api.ipify.org?format=json |
